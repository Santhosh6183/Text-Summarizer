# GenAI Text Summarizer

A simple AI-powered text summarization app using **Hugging Face BART model** and **Gradio**.

## Features

- Summarize text from **input box** or **.txt files**
- Light red input box, green output box, black text
- Works for long texts safely
- Easy-to-use web interface

## Installation

```bash
git clone <your_repo_url>
cd text_summarizer
pip install -r requirements.txt
python app.py
